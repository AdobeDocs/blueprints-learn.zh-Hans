# 主要业务目标

关键业务目标定义了组织寻求通过其数字体验计划实现的战略成果。 每个目标都映射到一个或多个[用例模式](/help/blueprints/use-case-patterns/overview.md)，这些用例模式描述如何实施Adobe Experience Platform和应用程序以交付这些结果。

使用下表确定与您的组织最相关的业务目标，然后按照链接了解支持每个目标的用例模式、KPI和实施方法。

## 收购与增长

以下目标侧重于扩大覆盖面和扩大客户群。

| 目标 | 描述 | 关键KPI |
| --- | --- | --- |
| [获取新客户](acquisition-growth/acquire-new-customers.md) | 通过有针对性的客户获取促销活动、相似受众和付费媒体优化来扩展客户群。 | 新客户、客户获取成本、潜在客户/潜在客户转化 |
| [提升商机开发](acquisition-growth/increase-lead-generation.md) | 通过表单、事件、内容和多渠道参与，为销售渠道生成更多符合条件的潜在客户。 | 潜在客户、每个商机的成本、商机转化 |
| [提高网站参与度](acquisition-growth/increase-website-engagement.md) | 通过相关体验，缩短网站停留时间、每次会话显示页面以及与Web内容的交互。 | 页面逗留时间、参与度、转化率 |

## 收入和盈利

以下目标专注于推动收入增长及增加交易价值。

| 目标 | 描述 | 关键KPI |
| --- | --- | --- |
| [提高转化率](revenue-monetization/increase-conversion-rates.md) | 提高完成所需操作（如购买、注册或表单提交）的访客和潜在客户的百分比。 | 转化率、潜在客户转化、每个潜在客户的成本 |
| [增加收入和销售额](revenue-monetization/increase-revenue-sales.md) | 通过优化的数字渠道、营销活动和客户历程推动总收入增长。 | — |
| [提高交叉销售和追加销售收入](revenue-monetization/drive-cross-sell-upsell-revenue.md) | 根据行为和购买历史，向现有客户推广补充性和优质产品或服务。 | 追加销售/交叉销售%，增量收入，客户存留期值 |
| [提高客户忠诚度和存留期值](revenue-monetization/increase-customer-loyalty-lifetime-value.md) | 通过忠诚度计划、奖励和个性化参与，深化客户关系并最大化长期价值。 | 客户存留期值、留存率、追加销售/交叉销售% |

## 成本和效率

以下目标侧重于降低成本和提高营销效率。

| 目标 | 描述 | 关键KPI |
| --- | --- | --- |
| [降低客户获取成本](cost-efficiency/reduce-customer-acquisition-cost.md) | 提高定位效率，抑制现有客户参与客户获取促销活动，并优化媒体支出。 | 客户获取成本、每个商机的成本、效率 |
| [优化营销支出和ROI](cost-efficiency/optimize-marketing-spend-roi.md) | 通过更好的定位、归因、受众抑制和预算分配提高营销投资回报。 | 成本节约、客户购置成本、增量收入 |
| [改进数据质量和管理](cost-efficiency/improve-data-quality-governance.md) | 确保数据干净、完整且合规，以实现准确定位、减少浪费和可靠的分析。 | 效率，成本节约 |
| [整合营销技术并使其现代化](cost-efficiency/consolidate-modernize-marketing-technology.md) | 通过迁移到统一、可扩展的平台，减少工具碎片化和技术债务。 | 成本节约、效率、上市速度 |

## 客户体验

以下目标侧重于在客户的整个生命周期中提供更好的体验。

| 目标 | 描述 | 关键KPI |
| --- | --- | --- |
| [提供个性化的客户体验](customer-experience/deliver-personalized-customer-experiences.md) | 根据个人偏好、行为和生命周期阶段定制内容、选件和消息。 | 参与度、转化率、客户满意度(CSAT) |
| [提高客户保留率](customer-experience/improve-customer-retention.md) | 透过价值导向型经验及持续培养关系，让现有客户持续参与及不断更新。 | 维系率、客户存留期值、参与度 |
| [改进客户入门](customer-experience/improve-customer-onboarding.md) | 通过优化的个性化欢迎体验和激活历程，加快新客户的价值实现。 | 参与度、维系率、转化率 |
| [恢复放弃的购物车和历程](customer-experience/recover-abandoned-carts-journeys.md) | 通过及时、个性化的跟进，重新吸引在购买、申请或注册流程中放弃的用户。 | 转化率、增量收入、参与度 |
