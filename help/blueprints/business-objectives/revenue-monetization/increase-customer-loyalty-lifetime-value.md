# 提高客户忠诚度和存留期价值

通过忠诚度计划、奖励和个性化参与，深化客户关系并最大化长期价值。 此目标侧重于通过始终如一的价值驱动型体验来建立持久的客户关系，从而推动重复购买、提高生命周期价值和品牌宣传。

## 关键绩效指标

以下KPI可衡量实现此目标的进度。

| KPI | 描述 |
| --- | --- |
| 客户存留期值 | 客户在整个关系中的总预测收入 |
| 保留 | 在规定期间内保持活跃的客户百分比 |
| 追加销售/交叉销售% | 购买其他或升级产品的现有客户的百分比 |
