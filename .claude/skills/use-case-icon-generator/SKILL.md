# 用例图标生成器

在Blueprint存储库中生成用例图标的详细图标规范和AI图像生成提示。

## 何时使用此技能

- 创建需要图标的新用例
- 由行业用例生成器技能调用以生成图标规范
- 用户询问创建、更新或替换用例图标
- 用户希望为新垂直行业生成一批图标

## 步骤1：收集输入

从用户或呼叫技能收集以下信息：

**必需：**
- **用例名称** — 易于用户识别的用例名称（例如，“放弃的购物车”、“商机评分”）
- **垂直行业** — 其中之一：汽车、b2b、金融服务、医疗保健、保险、媒体娱乐、零售、电信、旅游和酒店业
- **简要说明** — 1-2个描述用例功能的句子

**可选：**
- **首选的可视化隐喻或主题** — 如果用户对图标应描述的内容有特定想法

如果缺少任何必需的输入，请在继续之前询问用户。

## 第2步：阅读样式指南

阅读文件`references/icon-style-guide.md`（相对于该技能的目录）以加载完整的样式指南，包括：

- 技术规范
- 视觉样式指南
- 按行业划分的可视化隐喻示例
- 提示模板
- 现有图标清单

使用样式指南确保与现有图标一致。

## 步骤3：生成图标规范

生成图标规范的所有三个部分：

### A.文件规范

从用例名称和行业获取文件名和路径：

- **文件名：** `icon-{kebab-case-name}.png`
   - 将用例名称转换为kebab大小写（小写，空格使用连字符）
   - 示例：“放弃的购物车”变为`icon-abandoned-cart.png`，“交叉销售追加销售”变为`icon-cross-sell-upsell.png`
- **路径：** `/help/blueprints/industry-use-cases/assets/use-case-icons/{industry}/`
- **尺寸：** 1024 x 1024像素
- **格式：** PNG、8位RGB或RGBA
- **目标文件大小：** 900KB - 1.4MB

### B.图像生成提示

为AI图像生成工具创建详细的复制就绪提示。 提示必须：

1. **描述一个清晰的可视化隐喻** — 请选择一个能立即传达用例概念的清晰的可视化隐喻。 如果用户提供了首选的比喻，请使用该比喻。 否则，请根据样式指南示例和用例描述选择一个样式。

2. **指定艺术风格** — 包括以下风格指示：
   - 简洁的现代图标插图样式
   - 专业的企业设计美学
   - 粗体形状和简洁的线条
   - 抛光的、渲染的表面（不逼真）
   - 与统一设计系统一致

3. **包括技术规范：**
   - 正方形格式，1024x1024像素
   - 单个中心主题
   - 纯色或细微渐变背景
   - 主题和背景之间的高对比度

4. **强制小尺寸清晰度：**
   - 无任何文本或字母
   - 没有细微的细节或复杂的图案
   - 没有复杂背景或多元素场景
   - 清晰显示40px的粗体剪影
   - 即使没有颜色，也能实现强大的形状识别

5. **引导调色板：**
   - 专业的、适合公司的颜色
   - 生机勃勃但优雅（不霓虹灯或过度饱和）
   - 用于辅助功能的高对比度
   - 与同一垂直行业中的其他图标具有凝聚力

将提示构造为单个段落，可随时粘贴到任何图像生成工具中。

### C.目录引用代码片段

生成要在目录表中使用的HTML代码片段：

```
<img src="assets/use-case-icons/{industry}/icon-{name}.png" alt="{Alt Text}" width="40">
```

其中`{Alt Text}`是易于用户识别的用例名称。

## 步骤4：列出同一行业中的现有图标

选中样式指南的现有图标清单部分，并列出同一行业的所有当前图标。 这有助于用户：
- 在生成新图标时确保视觉一致性
- 避免复制已存在的图标
- 引用现有图标作为样式示例

## 指南

- **从不生成实际图像** — 此技能仅生成规范和提示。 用户必须使用外部图像生成工具。
- **每个用例一个图标** — 每个用例只获取一个图标。
- **检查重复项** — 如果清单中已存在相同或非常相似的图标，请警告用户。
- **优先考虑易读性** — 如果对详细隐喻和简单隐喻有疑问，请始终选择读数在40像素时读得很好的简单选项。
- **在提示中指定** — Vague提示会产生不一致的结果。 包括具体的视觉详细信息（例如，“一个内置两个彩色盒子的购物车”而不是“购物概念”）。
- **尽可能避免陈词滥调** — 尝试找到新鲜但仍可立即识别的视觉隐喻。 每个“智能”用例的灯泡都会重复出现。
